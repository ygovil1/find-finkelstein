----------------------------------------------------------------
Trapped in Firestone - Escape the Room
By Yashodhar Govil, Khanh Vu and Urvashi Uberoy
----------------------------------------------------------------

This directory contains the following folders/files:

1. index.html - contains all the Three.js code for scene setting
                and object handling.
2. js/Form.js - contains the code for the inventory and hints

3. img/ - contains all the images used for the hints.

4. textures/ - contains images for the carpet textures, etc.


----------------------------------------------------------------
Run Instructions
----------------------------------------------------------------
To run the game, make sure you're in find-finkelstein and then enter:

$ python3 -m http.server

The game should load on: http://localhost:8000/
